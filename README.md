## 📦 Installation (Termux)

```bash
pkg install python git
pip install requests beautifulsoup4
git clone https://github.com/darkvibez456/python-hacking-scripts
cd python-hacking-scripts
python3 tools/scanner.py
```

## 📦 Installation (Linux)

```bash
git clone https://github.com/darkvibez456/python-hacking-scripts
cd python-hacking-scripts
pip install -r requirements.txt
python3 tools/scanner.py
```

## ⚡ Features

- 🚀 **Fast Port Scanning**: Multi-threaded port discovery.
- 🛡️ **XSS Detection**: Automated vulnerability auditing.
- 📱 **Termux Optimized**: Fully compatible with Android Termux.
- 🐧 **Linux Ready**: Works on all major Linux distributions.
- 🎨 **Pro UI**: Colorful command-line interface with ASCII art.
