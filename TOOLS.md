# Tools List

This document provides an overview of the various Python scripts and tools available in this repository, categorized by their primary function.

## Information Gathering

-   **`scanner.py`**: A basic network scanner to identify open ports and services.
-   **`whois_lookup.py`**: Performs WHOIS lookups for domain information.

## Vulnerability Analysis

-   **`xss_detector.py`**: Detects potential Cross-Site Scripting (XSS) vulnerabilities in web applications.
-   **`sql_injector.py`**: A simple tool to test for SQL injection vulnerabilities.

## Exploitation

-   **`reverse_shell.py`**: Creates a reverse shell for remote access (for educational purposes only).

## Post-Exploitation

-   **`keylogger.py`**: A basic keylogger to capture keystrokes (for educational purposes and authorized testing only).

## Password Cracking

-   **`bruteforce_ftp.py`**: Attempts to bruteforce FTP login credentials.

## Other Utilities

-   **`mac_changer.py`**: Changes the MAC address of a network interface.

**Note:** This list will be updated as more tools are added to the repository. Always use these tools responsibly and ethically.
